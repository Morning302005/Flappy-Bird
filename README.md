# Flappy-Bird
A polished 2D side-scroller built with Python and Pygame featuring parallax scrolling, custom particle physics, and dynamic neon rendering

A fast-paced "Flappy Bird" style survival game built using **Python** and **Pygame**. Navigate the bird through neon fire pillars while managing gravity and momentum.

## Features
* **Parallax Background:** Dual-layer scrolling for depth.
* **Particle System:** Exploding effect upon collision.
* **Neon Graphics:** Custom-drawn fire with glow effects.
* **Audio:** Background music and sound effects.

## How to Run
1. **Install Python 3.x**
2. **Install Pygame**:
   ```bash
   pip install pygame

## Controls
* SPACE: Jump / Fly
* R: Restart (after Game Over)
* ESC: Pause/Unpause
